# Facebook Manual Auto Poster

هذا المشروع يشغل بوت Node.js للنشر التلقائي إلى **Facebook Page** بدون ذكاء اصطناعي.

الفكرة الآن بسيطة:

- تلصق نصًا كبيرًا في الداشبورد
- كل جزء بين `@` و `@` يتحول إلى منشور مستقل
- هذه المنشورات تدخل إلى طابور النشر
- البوت ينشر أول منشور في الطابور حسب الوقت الذي تحدده

## المتغيرات

ضع هذه القيم في `.env` أو Railway:

- `FB_PAGE_ID`
- `FB_PAGE_ACCESS_TOKEN`
- `POST_INTERVAL_MINUTES`
- `TIMEZONE`

اختياري:

- `BASE_URL`
- `STATE_DIR`
- `FB_APP_ID`
- `FB_APP_SECRET`

## التشغيل

```bash
npm install
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## الدخول

- الصفحة الرئيسية `/` تعرض شاشة الكود مباشرة
- كود الدخول الحالي هو `5598`
- بعد الدخول ستذهب إلى الداشبورد

## صفحات الداشبورد

- `نظرة عامة`
- `التحكم في الوقت`
- `المنشور التالي`
- `المنشورات التي تم نشرها`
- `الأشخاص المتفاعلون`
- `إدارة المنشورات`

## طريقة إضافة المنشورات

في صفحة `إدارة المنشورات`:

1. الصق نصًا كبيرًا في مربع النص
2. اجعل كل منشور بين `@` و `@`
3. اضغط `إضافة المنشورات`

مثال:

```text
@المنشور الأول@

أي كلام زائد هنا لن يؤخذ

@المنشور الثاني
يمكن أن يكون على أكثر من سطر@

@المنشور الثالث@
```

النتيجة:

- كل جزء بين `@` و `@` يدخل كمنشور مستقل
- يظهر داخل بطاقة مميزة
- لكل بطاقة زر `x` للحذف
- بجانب الزر يظهر رقم المنشور
- يمكنك إضافة عدد غير منتهٍ من المنشورات على دفعات

## النشر

- صفحة `المنشور التالي` تعرض أول منشور موجود في الطابور مباشرة
- صفحة `التحكم في الوقت` تتحكم في الفاصل بين المنشورات
- عند تشغيل البوت، إذا كان الطابور غير فارغ، يبدأ النشر تلقائيًا حسب الجدولة
- يمكنك أيضًا تجربة نشر يدوي من:

```text
/run-once
```

## ملاحظات

- المنشورات المخزنة والحالة تُحفظ في `data/state.json`
- المنشورات المنشورة تُعرض في صفحة `المنشورات التي تم نشرها`
- الأشخاص المتفاعلون يتم جلبهم من تعليقات أحدث المنشورات المنشورة
- هذا الإصدار لا يستخدم Gemini ولا أي مزود ذكاء اصطناعي
